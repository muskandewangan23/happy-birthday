# 🎂 Happy Birthday, Arya ❤️

A small interactive birthday website made with lots of love, memories, and a little bit of JavaScript.

## 🌐 Live Website

👉 **[Open the Birthday Website ❤️](https://muskandewangan23.github.io/happy-birthday/)**

## ✨ Features

- 🎵 Instrumental background music
- 💻 Animated loading screen
- ❤️ Memory timeline
- 📸 Interactive photo gallery
- 💌 Animated heartfelt letter
- 💖 Secret message
- 🎂 Birthday ending page
- 😂 Surprise fun video at the end

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
